{% comment %} The shared calendar grid. Included by both day.html and week.html. The view hands over uniform column dicts — head, blocks, click_date, click_param, clickable, show_now, is_unassigned — so this template branches on booleans and never has to work anything out. Every branch a column can take is decided in Python, where a missing key is an error rather than an empty string. GEOMETRY. Every pixel value is an INTEGER computed in the view. A float here would render with the active locale's decimal separator — "112,666" under a non-English locale — making the surrounding calc() invalid and snapping every block to the top of its column. Events sit in `.calendar-column-body`, NOT directly in `.calendar-column`: the column head is sticky but still IN FLOW, so a column-relative offset would put every block one head-height above its own gridline. {% endcomment %}
