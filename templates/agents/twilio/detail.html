About these credentials

Credentials are per location, because each site may run its own Twilio subaccount. The auth token is encrypted at rest and is never shown again once saved — to change it, paste a new one.

The inbound number

One number, one location, across the whole platform. An inbound call carries no session, so the number dialled is what tells the runtime which business and which site is being called.
